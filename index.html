Open Decisions — Source & Destination (for pilot)

Source — where to extract?

  • Which platforms: YouTube, X, others?
  • Official channels, live videos, comments, replies, hashtags?
  • Who provides the entity registry and API access?
  • Live polling cadence vs push webhooks?

Destination — where to deliver?

  • S3 data lake paths for raw vs enriched JSON?
  • Pub/sub topics (EventBridge / SQS) for real-time events?
  • Live dashboard, alerts, or other agents subscribing?
  • Schema contract for downstream consumers?
