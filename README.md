# Engagement Analytics — interactive demo

Standalone HTML demo.

## What it shows

- **7-step real-time agent pipeline:** Source → Listener → Gather → Normalize → Analyze → Orchestrate → Publish
- Animated flow with **Start / Pause / Reset**
- 4 simulated **live windows** (pre-match, goal spike, VAR negative, YouTube chat)
- Window inspector + query (`negative`, `goal`, `VAR`, `YouTube`, `spike`)
- Live insight JSON output
- **Open decisions**
